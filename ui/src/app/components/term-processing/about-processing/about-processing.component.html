Про проєкт UkrVectōrēs

Сервіс UkrVectōrēs обчислює семантичні відношення між сутностями української мови в рамках обраної дистрибутино-семантичної моделі векторного представлення сутностей.

Корисні посилання:

Gensim — програмна бібліотека з відкритим початковим кодом для комплексного моделювання мовного векторного простору. Туторіал з tensorflow.org — на тему "Векторне представлення слів / Навчання вкладень слів". fastText — програмна бібліотека з відкритим початковим кодом для навчання вкладень слів та класифікування тексту.
